PART 1: PRODUCT REQUIREMENTS DOCUMENT
1.1 Executive Summary
InspirePost is an automated system for generating and publishing inspirational quote images to Instagram. The system combines curated wisdom quotes with AI-generated original content, creates visually appealing nature-themed images with embedded text using Google's Gemini 3 Pro Preview API (Nano Banana Pro), and manages a 7-day content queue with scheduled posting capabilities.
1.2 Product Vision
Build an automated, approval-based workflow that generates high-quality inspirational content, creates beautiful nature-themed images, maintains a rolling 7-day content preview on a Vercel-hosted website, and publishes approved content to Instagram at optimal engagement times.
1.3 Target Audience
� Primary: Instagram users seeking daily inspiration and motivation
� Secondary: Personal development enthusiasts, mindfulness practitioners
� Theme: General life wisdom (philosophy, motivation, mindfulness, success)
1.4 Content Strategy
1.4.1 Quote Sources (Hybrid Approach)
Curated Quotes (60%):
� ZenQuotes API - Free tier with attribution, excellent quality inspirational quotes
� Quotable API (quotable.io) - Open source, 180 requests/minute, filterable by tags
� API Ninjas Quotes - Categorized quotes (wisdom, inspirational, success, philosophy)
AI-Generated Original Content (40%):
� Generated using Claude API or similar LLM
� Themes: mindfulness, gratitude, resilience, growth mindset, inner peace
� Style: Concise (under 150 characters), original, non-attributed
1.4.2 Posting Schedule
� Frequency: 3 posts per week
� Recommended Days: Tuesday, Wednesday, Thursday (highest engagement per research)
� Optimal Times: 9:00 AM - 12:00 PM local time (late morning peak engagement)
� Content Queue: Always maintain 7 days of approved content ready for posting
1.5 Visual Design Requirements
1.5.1 Image Specifications
� Format: JPEG (required by Instagram API)
� Aspect Ratio: 4:5 portrait (1080x1350px) - optimal for Instagram feed
� Background Theme: Nature-inspired (forests, mountains, ocean, sunrise/sunset, meadows)
1.5.2 Branding Guidelines
� Font: Consistent across all posts (recommend: Playfair Display for quotes, Montserrat for attribution)
� Color Palette: Earthy tones - forest greens, sky blues, warm sunset oranges, soft neutrals
� Text Placement: Centered with semi-transparent overlay for readability
� Watermark: Subtle account handle in corner (optional)
1.6 Functional Requirements
1. Quote Generation: System shall fetch curated quotes from APIs and generate original AI content
2. Image Generation: System shall use Gemini 3 Pro Preview API to create nature-themed images with embedded quote text
3. Preview Website: System shall display 7-day content queue on davetest.drewhowlett.com
4. Approval Workflow: User must approve content before it enters the publishing queue
5. Scheduling: System shall schedule posts for optimal engagement times
6. Instagram Publishing: System shall post approved content via Instagram Content Publishing API
7. Error Handling: System shall alert user for manual intervention on any failures
8. Queue Management: System shall auto-replenish queue to always show 7 future days
1.7 User Workflow
1. System auto-generates quote + image candidates
2. Content appears on preview website (davetest.drewhowlett.com)
3. User reviews and approves/rejects each piece of content
4. Approved content enters scheduled queue
5. Scheduler posts to Instagram at optimal times
6. User receives alerts on any failures for manual intervention


PART 2: TECHNICAL SPECIFICATION
2.1 System Architecture
Architecture Pattern: Serverless with local scheduler (MVP) ? GitHub Actions/Vercel Cron (Production)
Core Components:
� Quote Service - Fetches/generates inspirational content
� Image Generator - Creates visuals via Imagen 3 API
� Content Manager - Handles queue, approval state, scheduling
� Preview Website - Next.js app on Vercel showing 7-day queue
� Instagram Publisher - Posts approved content via API
� Notification Service - Alerts on failures
2.2 API Integrations
2.2.1 Google Gemini 3 Pro Preview API (Nano Banana Pro)
Purpose: Generate nature-themed images with quote text embedded directly in the image
Model: gemini-3-pro-image-preview
Cost: ~$0.04 per image
Setup: Google AI Studio ? Create Project ? Generate API Key
Endpoint: https://generativelanguage.googleapis.com/v1beta/models/gemini-3-pro-image-preview:generateContent
Image Generation Strategy:
� Generate complete image with text using Gemini 3 Pro Preview
� Text is rendered directly by the AI model (no separate overlay needed)
� Resize to 1080x1350px using Sharp
� Export as JPEG
2.2.2 Instagram Content Publishing API
API Type: Instagram Platform API (launched July 2024)
Key Benefit: Direct Instagram login without Facebook Page requirement
Supported Account Types: Business and Creator accounts
Rate Limit: 25 posts per 24 hours
Image Format: JPEG only
Required Scopes:
� instagram_business_basic
� instagram_business_content_publish
Publishing Flow:
� Step 1: Create media container (POST /{ig-user-id}/media)
� Step 2: Wait for Instagram to process (~10 seconds)
� Step 3: Publish container (POST /{ig-user-id}/media_publish)

2.5 Prerequisites Setup Guide
2.5.1 Instagram Account Setup
� Open Instagram app ? Settings ? Account ? Switch to Professional Account
� Choose 'Creator' account type (recommended for content creators)
� Select appropriate category (e.g., 'Digital Creator', 'Personal Blog')
� Complete profile setup
2.5.2 Meta Developer App Setup
� Go to developers.facebook.com ? Create App
� Select 'Business' as app type
� Add Instagram API product to your app
� Configure OAuth redirect URIs
� Request instagram_business_basic and instagram_business_content_publish permissions
� Complete App Review process for production access
2.5.3 Google AI Studio Setup (Imagen 3)
� Go to aistudio.google.com
� Sign in with Google account
� Create new project or select existing
� Navigate to API Keys ? Create API Key
� Enable billing (required for image generation)
� Store API key securely in environment variables
